# 🤖 Настройка Telegram Web App

## ✅ Деплой завершён

## 📋 Настройка в BotFather

Откройте @BotFather в Telegram и выполните следующие шаги:

### 1. Создание Web App

Отправьте команду:
```
/newapp
```

BotFather попросит:
1. **Выбрать бота** — выберите вашего бота
2. **Название приложения** — введите: `LifeOS`
3. **Описание** — введите: `AI Life Engineering System`
4. **Web App URL** — вставьте: `https://life-os-seven-khaki.vercel.app`
5. **Короткое название кнопки** — введите: `LifeOS`

### 2. Настройка Menu Button

Отправьте команду:
```
/mybots
```

1. Выберите вашего бота
2. **Bot Settings** → **Menu Button** → **Configure Menu Button**
3. Отправьте URL: `https://life-os-seven-khaki.vercel.app`
4. Введите название кнопки: `Открыть LifeOS`

### 3. Готово!

Теперь пользователи могут:
- Нажать на кнопку меню в боте
- Или перейти по прямой ссылке: `t.me/YourBotName/app`

---
