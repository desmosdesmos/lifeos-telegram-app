# Настройка Qwen AI API

## Что изменилось

Раньше использовался **GigaChat (Сбер)**, который работал только из РФ.  
Теперь используется **Qwen (Alibaba)** — работает глобально, без географических ограничений.

## Преимущества Qwen

✅ **Работает из любой страны** — нет проблем с Vercel (США)  
✅ **Текстовые модели** — qwen-plus, qwen-turbo  
✅ **Vision (анализ фото)** — qwen-vl-plus  
✅ **Бесплатные токены** — новый пользователи получают гранты  
✅ **OpenAI-совместимый API** — легко интегрировать  

## Получение API ключа

### Шаг 1: Зарегистрируйтесь на Alibaba DashScope
1. Перейдите на https://dashscope.console.aliyun.com/
2. Войдите через Alibaba Cloud аккаунт (или создайте новый)
3. Пройдите верификацию (требуется телефон)

### Шаг 2: Получите API ключ
1. Перейдите в **Console** → **API Keys**
2. Нажмите **Create API Key**
3. Скопируйте ключ (начинается с `sk-`)

### Шаг 3: Активируйте бесплатные токены
1. Новые пользователи получают **бесплатные токены** для тестирования
2. Проверьте доступные квоты в разделе **Quota Center**

## Настройка на Vercel

### Шаг 1: Зайдите в проект на Vercel
1. Откройте https://vercel.com/dashboard
2. Выберите ваш проект "LIFE OS"

### Шаг 2: Добавьте Environment Variable
1. Перейдите в **Settings** → **Environment Variables**
2. Добавьте переменную:

   **QWEN_API_KEY**
   - Value: `sk-fe7020d75d9d41dab3449a42e1534cfd` (или ваш ключ)
   - Environments: ✅ Production, ✅ Preview, ✅ Development

### Шаг 3: Задеплойте изменения
1. Вернитесь на вкладку **Deployments**
2. Нажмите **Redeploy** на последнем деплое
3. Дождитесь завершения деплоя

## Локальная разработка

### Шаг 1: Создайте `.env` файл
Скопируйте `.env.example` в `.env`:
```bash
cp .env.example .env
```

### Шаг 2: Добавьте API ключ
```env
QWEN_API_KEY=sk-fe7020d75d9d41dab3449a42e1534cfd
```

### Шаг 3: Запустите проект
```bash
npm install
npm run dev
```

## Troubleshooting

### Ошибка "Qwen API key not configured"
- Проверьте, что переменная `QWEN_API_KEY` добавлена в Vercel
- Убедитесь, что имя точное (без опечаток)
- Сделайте Redeploy после добавления переменной

### Ошибка "Auth failed: 401"
- Проверьте правильность API ключа
- Убедитесь, что ключ активен (не истёк срок действия)
- Проверьте, что аккаунт верифицирован

### Ошибка "Cannot connect to Qwen API"
- Проверьте интернет-соединение
- Qwen API может быть временно недоступен
- Проверьте логи в Vercel Dashboard → Functions → /api/chat

### Ошибка "Insufficient quota"
- Проверьте доступные квоты в DashScope Console
- При необходимости пополните баланс или дождитесь обновления квоты
