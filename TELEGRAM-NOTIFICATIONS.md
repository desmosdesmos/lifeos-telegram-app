# Настройка уведомлений о входах пользователей

## 1. Создайте Telegram бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Придумайте имя боту (например, `Life OS Notifications`)
4. Придумайте юзернейм боту (должен заканчиваться на `bot`, например `lifeos_notify_bot`)
5. BotFather выдаст вам **API токен** (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

## 2. Настройте файл .env

Откройте файл `.env` и заполните:

```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_ADMIN_ID=478799066
```

Замените `TELEGRAM_BOT_TOKEN` на токен, который вы получили от BotFather.

## 3. Запустите бота

После первого запуска приложения, когда вы зайдете в бота/приложение, вам придет уведомление вида:

```
🔔 Новый вход в приложение

👤 Пользователь: Иван Иванов @ivanov
🆔 ID: 123456789
🌐 Язык: ru
📅 Время: воскресенье, 29 марта 2026 г., 14:30:00
```

## Важно

- Уведомления отправляются только при загрузке приложения в Telegram WebApp
- Каждое уведомление отправляется один раз за сессию (при загрузке страницы)
- Если вы не получили уведомление — проверьте консоль браузера на наличие ошибок
